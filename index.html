All tools are based on JavaScript and will work completely on your machine! No Mac Adress or other information is sent to any server.

>>> Gira .hslx AES-Decrypter This tool will decrypt HomeServer logic blocks that have been bound to a specific Serial Number. Those branded logic blocks are shipped as a .hslz file. You can open them using WinRar. Inside there is a encrypted .hslx file and a generic .hsl file. This tool will decrypt and combine the both files into a single unencrypted .hsl file which can be used by other HomeServers.
>>> Grot Keygen This tool will generate license keys for products of Grot Automation. Backup download of the Plugins and logic blocks here!
>>> Grot Keychecker This tool is mainly for testing purposes or validation of legally purchased keys.
>>> TecSupport Keygen This tool will generate keys for products of TecSupport. Backup download of the logic blocks here!
>>> TecSupport HPV This tool will generate a license for TecSupport HPV. Backup download of the software here!
>>> TecSupport TecTool This tool will generate a software license for TecSupport TecTool. Backup download of the software: Windows 7-11 / Windows XP!
>>> HagerCAD A plain KeyGen for HagerCad.
>>> Paid .etsapps This is a collection of skimmed .etsapps from the store. With a cracked ETS5/6 those etsapps work without license. There are some .etsapps included that are not available on the store. Use especially DeviceEditorApp with caution!
